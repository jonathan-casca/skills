## Install in Cursor

1. Open **Customize** → **Rules**
2. Click **Add Rule** → **Remote Rule (GitHub)**
3. Paste: `https://github.com/jonathan-casca/skills`
4. Reload Agent chat and run `/plan-team <task>`

You can also copy a skill folder into `~/.cursor/skills/` or `<repo>/.cursor/skills/`.
