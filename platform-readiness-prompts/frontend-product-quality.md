# Frontend Product Quality Prompts

```text
Audit the frontend for production UX gaps. Look for missing loading states, error states, empty states, permission states, retry affordances, accessibility issues, and mobile responsiveness.
```

```text
Find every async UI flow and verify it handles loading, success, validation failure, network failure, unauthorized, forbidden, and empty states.
```

```text
Review forms for production readiness: validation, disabled states, duplicate submit prevention, optimistic updates, error display, accessibility, and data loss prevention.
```

```text
Audit this UI for accessibility issues: keyboard navigation, focus states, ARIA, semantic HTML, color contrast, modals, forms, and screen reader behavior.
```
