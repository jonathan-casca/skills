# Product Operations Prompts

```text
Audit this app for product operations readiness. Include feature flags, analytics events, lifecycle emails, notifications, support handoff, admin overrides, customer segmentation, onboarding, offboarding, and internal reporting.
```

```text
Create an analytics instrumentation plan. Define the core user journeys, events, properties, identity model, privacy rules, funnel metrics, activation metrics, retention metrics, and dashboards needed to understand product health.
```

```text
Review notification and email workflows. Check unsubscribe rules, delivery failure handling, retries, user preferences, template testing, localization if relevant, PII exposure, auditability, and support visibility.
```

```text
Design feature flag governance for this product. Include naming, ownership, rollout controls, expiration dates, cleanup process, per-tenant flags, kill switches, audit logs, and tests that cover both flag states.
```

```text
Audit onboarding and offboarding flows. Identify missing setup states, empty states, sample data needs, invite flows, permission handoffs, account closure, data export, and support escalation paths.
```

```text
Create a product health dashboard plan. Include activation, retention, conversion, task completion, error rates, latency, support tickets, billing health, usage by tenant, and signals that product quality is degrading.
```
