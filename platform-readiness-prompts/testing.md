# Testing Prompts

```text
Create a test strategy for this app. Identify the highest-risk flows and recommend unit, integration, and e2e tests. Include exact test cases and where they should live.
```

```text
Find business logic that has no tests. Prioritize by user impact and complexity. Write a plan to add focused regression coverage.
```

```text
Audit existing tests for false confidence. Find tests that overmock, assert implementation details, miss failure cases, or do not cover real user workflows.
```

```text
Generate a test matrix for roles, permissions, tenant boundaries, error states, empty states, and critical workflows.
```
